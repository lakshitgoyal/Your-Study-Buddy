# Your Study Buddy  

> An AI-powered learning assistant designed to help students study smarter, faster, and more effectively.

---

## Overview  

**Your Study Buddy** is a smart academic companion that helps students enhance their learning experience through AI-powered assistance.  

It is built to make studying more interactive, structured, and efficient by generating high-quality academic content on demand.

Whether you're preparing for exams, revising concepts, or exploring new topics, this tool acts as your personal study assistant.

---

## Features  

- Topic-based note generation  
- Clear and simplified concept explanations  
- Automatic practice question generation  
- Smart topic summarization  
- AI-powered academic assistance  
- Fast and responsive user experience  
- Designed specifically for students

## Installation  

Clone the repository:

```bash
git clone https://github.com/lakshitgoyal/Your-Study-Buddy.git
```

Navigate into the directory:

```bash
cd Your-Study-Buddy
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```

---

## Usage  

1. Enter the topic you want to study.  
2. Generate structured notes or explanations.  
3. Review and understand the AI-generated content.  
4. Practice using generated questions for better retention.
